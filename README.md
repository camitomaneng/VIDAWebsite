# VIDAWebsite

Landing page for the VIDA app.

## Preview locally

You can open the static `index.html` directly in a browser or serve it locally:

1. From the project root, start a simple server:

   ```bash
   python -m http.server 3000
   ```

2. Visit http://localhost:3000 in your browser to view the page.

Stop the server with `Ctrl+C` when you're done.
